# NovaMind Chat UI
A modern, responsive chat interface inspired by Claude and ChatGPT.

## Built With
- HTML5 (semantic markup)
- CSS3 (custom properties, flexbox, animations)
- JavaScript (ES6+)
- jQuery 3.7.1
- Bootstrap 5.3.3
- Font Awesome 6.5.1
- Google Fonts (Syne + DM Sans)

## How to Run
1. Unzip the folder
2. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
3. No server or installation needed — fully client-side
