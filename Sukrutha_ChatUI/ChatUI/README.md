# NovaMind Chat UI

A modern, responsive chat interface inspired by Claude and ChatGPT.

## Built With
- HTML5 (semantic markup)
- CSS3 (custom properties, flexbox, animations)
- JavaScript (ES6+)
- jQuery 3.7.1
- Bootstrap 5.3.3
- Font Awesome 6.5.1
- Google Fonts (Syne + DM Sans)

## Features
- ✅ Welcome screen with 4 suggestion cards
- ✅ User and AI message bubbles (visually distinct)
- ✅ Animated typing indicator (bouncing dots)
- ✅ Auto-resizing textarea input
- ✅ Enter to send, Shift+Enter for newline
- ✅ Sidebar with chat history (260px fixed)
- ✅ New Chat button
- ✅ Dark/Light mode toggle
- ✅ Typewriter animation for AI responses
- ✅ Message formatting (bold, italic, code blocks)
- ✅ Export chat as .txt file (Blob API)
- ✅ Custom scrollbar styling
- ✅ Mobile responsive (hamburger menu + slide-in sidebar)
- ✅ Responsive from 320px to 1920px

## How to Run
1. Unzip the folder
2. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
3. No server or installation needed — fully client-side
